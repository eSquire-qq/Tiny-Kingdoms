# Tiny Kingdoms

**Tiny Kingdoms**! Це 2D стратегія в реальному часі (RTS), яка складається з двох частин:
1. **`rts-game`** — Ігровий клієнт, розроблений на двигуні Unity.
2. **`rts-server`** — Авторитативний ігровий сервер на Java (Spring Boot) з інтегрованим TCP-сервером (порт 7777).

---

## Як запустити проект локально

Все налаштовано так, щоб мінімізувати проблеми з різними версіями Java чи Unity. Вам не потрібно встановлювати та налаштовувати локальну базу даних вручну.

### Попередні вимоги
Перед запуском переконайтеся, що у вас встановлені:
* **[Git](https://git-scm.com/)**
* **[Docker Desktop](https://www.docker.com/products/docker-desktop/)** (для автоматичного підняття MySQL)
* **[Unity Hub](https://unity.com/download)**
